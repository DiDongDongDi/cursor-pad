# Cursor Pad

手机端桌面浏览器模拟器：强制桌面视口渲染，配合虚拟鼠标与触控板手势，在横屏模式下更接近 PC 浏览器体验。

## 功能（MVP）

- 默认横屏启动
- 桌面 User-Agent 与 viewport 注入
- 虚拟鼠标光标（相对滑动移动）
- 单击 / 双击 / 长按右键 / 双指滚动
- 左键按住 + 拖动选字，长按复制
- 地址栏、前进、后退、刷新、主页

## 运行

```bash
flutter pub get
flutter run
```

## 项目结构

- `lib/features/browser/` — WebView 与浏览器壳
- `lib/features/cursor/` — 虚拟光标
- `lib/features/input/` — 触控板手势
- `assets/js/` — 注入脚本（桌面模式、鼠标桥接）

## 手势说明

| 手势 | 行为 |
|------|------|
| 单指拖动 | 移动虚拟鼠标 |
| 单指轻触 | 左键点击 |
| 双击 | 双击 |
| 长按 | 右键 |
| 双指拖动 | 页面四向滚动 |
| 工具栏「左键按住」→ 单指拖动 → 再点关闭 | 选中页面文字 |
| 选中文字后长按 | 右键菜单复制 |
